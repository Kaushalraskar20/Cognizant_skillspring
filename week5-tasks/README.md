# Week 5 — React Fundamentals (HOL 1–13)

| HOL | App Name | Key Concept |
|-----|----------|-------------|
| 1  | myfirstreact          | First React component, JSX |
| 2  | StudentApp            | Class components, multiple components |
| 3  | scorecalculatorapp    | Function components, Props, CSS stylesheet |
| 4  | blogapp               | Lifecycle: componentDidMount, Fetch API, componentDidCatch |
| 5  | cohortdashboard       | CSS Modules, conditional inline styling |
| 6  | TrainersApp           | React Router v6: BrowserRouter, Routes, Route, Link, useParams |
| 7  | shoppingapp           | Props — passing data parent → child |
| 8  | counterapp            | State — setState, class component state |
| 9  | cricketapp            | ES6: map, filter, arrow functions, destructuring, spread |
| 10 | officespacerentalapp  | JSX elements, attributes (img src), objects, conditional CSS |
| 11 | eventexamplesapp      | Events: onClick, multiple methods, arguments, SyntheticEvent |
| 12 | ticketbookingapp      | Conditional rendering: if/else, ternary, && |
| 13 | bloggerapp            | All 4 conditional rendering techniques combined |


## How to Run Any HOL

```bash
cd HOL1-myfirstreact    # (or any HOL folder)
npm install
npm start               # opens http://localhost:3000
```
