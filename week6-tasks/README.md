# Week 6 — Advanced React + GIT + CI/CD (HOL 14–19)

| HOL | App Name | Key Concept |
|-----|----------|-------------|
| 14 | employee-context  | Context API: createContext, Provider, useContext (no prop drilling) |
| 15 | ticketraisingapp  | Controlled forms: handleChange, handleSubmit, form reset |
| 16 | mailregisterapp   | Form validation: real-time (onChange) + submit (onSubmit) |
| 17 | fetchuserapp      | REST API: fetch + componentDidMount + async data display |
| 18 | cohort-testing    | Unit testing: @testing-library/react, describe, test, expect |
| 19 | gitclientapp      | Jest mocking: jest.mock('axios'), mockResolvedValue |

---

## How to Run Any HOL

```bash
cd HOL1-myfirstreact    # (or any HOL folder)
npm install
npm start               # opens http://localhost:3000
```

## How to Run Tests (HOL 18 & 19)

```bash
cd HOL18-cohort-testing
npm install
npm test

cd HOL19-gitclientapp
npm install
npm test
```
